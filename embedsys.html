  • videogame_assetAdventure Game

    Languages Used: C++

    Team Size: 1

    Role: Programmer

    SimpleVania is a video game implemented on a microcontroller. Simplevania resembles the gameplay style of the Nintendo classic, CastleVania.

    The player controls the character with a Super Nintendo Controller connected to the microcontroller. After receiving input from the controller the player will move across each level which appears on the 8x8 LED matrix. Additional information is displayed on the LCD screen located above the LED matrix. Game logic includes character movement, synchronized updates of the LCD screen with the game as well as a delay on the playerâ€™s attack, just like the original Castlevania game.

    Levels were programmed by defining different values as tiles to be used within an array that would be referred to when displaying to the LED matrix. For every movement a buffer array would be updated.

  • storageStorage System

    Languages Used: C++

    Team Size: 2

    Role: Programmer

    The project is a lockbox system.

    The system starts up once motion is detected with a start message displayed on an LCD screen. Navigation is done with a game controller. Once a box is selected a signal is sent over to the specified box which a stepper motor unlocks. After an item is stored or retrieved the box can be closed by pressing the START button. If no activity is detected then the stepper motor locks the box that was opened.

    The program controls two microcontrollers at the same time.
